===================
silva.pas.kerberos5
===================

This extension provides a PAS plugin to authenticate against a
Kerberos 5 server. It order to install this extension, you must first
install MIT implementation of Kerberos.

It is designed to be used in `Silva`_.

Code repository
===============

You can find the code of this extension in Git:
https://github.com/silvacms/silva.pas.kerberos5

.. _Silva: https://silvacms.org
